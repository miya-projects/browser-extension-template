## 一个开箱即用的Chrome扩展模板

* 基于angular，rollup
* 提供SPA应用路由监听，消息通讯Demo,以及常用功能封装


## 快速开始
1. 安装依赖 `npm install`
2. 运行 `npm run watch`
3. 加载已解压的扩展程序，选择`dist/browser`目录
4. 打包 `npm run build`


> 文档： [https://developer.chrome.google.cn/docs/extensions/develop?hl=zh-cn](https://developer.chrome.google.cn/docs/extensions/develop?hl=zh-cn)
